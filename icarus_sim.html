Project Icarus Flight Simulations

Overview

This project focuses on simulating the flight dynamics and control algorithms of Project Icarus. Current focus on Python Simulations and integrating simple controls to smooth flight profile.

Simulation Gallery

Current Progress

Python simulation was successful of a simple three phase flight. The next step will be adding PID to go from ground -> 10m -> soft landing.

  • Test simple three-phase flight on Icarus platform.

Technical Details

  • Focus Areas: Dynamics modeling, state estimation, autonomous control testing
