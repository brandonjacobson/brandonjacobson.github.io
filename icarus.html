Project Icarus

Overview

Project Icarus is a custom-built quadcopter designed to explore flight dynamics, control theory, and onboard automation. The drone integrates a Pixhawk flight controller running PX4, custom power distribution, and telemetry logging through MAVSDK. It serves as a hands-on platform for testing real-time control algorithms, state estimation, and autonomous flight modes.

Media Gallery

Current Progress

Currently tuning the quadcopter’s PID controllers for stable hover and attitude control. Working on integrating onboard sensors for real-time state estimation with Kalman filtering, and building autonomy scripts in MAVSDK for multi-stage flight sequences such as takeoff, waypoint navigation, and landing.

Technical Details

  • Flight Controller: Pixhawk running PX4 firmware
  • Programming: MAVSDK (Python/C++), PID and Kalman filter implementation
  • Hardware: Custom frame, ESCs, telemetry radio, LiPo battery, onboard sensors
