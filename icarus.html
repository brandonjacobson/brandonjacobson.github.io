Project Icarus

Overview

Project Icarus is a custom-built quadcopter designed to explore flight dynamics, control theory, and onboard automation. The drone integrates a Pixhawk flight controller running PX4, custom power distribution, and telemetry logging through MAVSDK. It serves as a hands-on platform for testing real-time control algorithms, state estimation, and autonomous flight modes.

Media Gallery

Current Progress

Currently working on finding a new functional Flight Controller to pair with the Raspberry Pi 4B. The previous flight controller had issues with its RAM chip, not storing parameters for flight correctly on power cycle.

Upcoming Goals

Integrate Raspberry Pi with Flight Controller. Fly first manual radio test flight to confirm hardware and assembly.

Technical Details

  • Flight Controller(Previous): Pixhawk running PX4 firmware
  • Programming: MAVSDK (Python/C++), PID and Kalman filter implementation coming soon
  • Hardware: Custom frame, 35A individual ESCs, FS-IA6B/FlySky FS-i6 radio, 2300mAh 4S 75C LiPo battery, onboard sensors
